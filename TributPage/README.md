## 📌 Project Overview

This project is a Tribute Page dedicated to **Cristiano Ronaldo (CR7)**, celebrating his life, career, achievements, and legacy in football.

Developed as part of the **Oasis Infobyte Internship Program (OIBSIP)**, the page features a modern dark editorial design inspired by Ronaldo's iconic red and gold branding.

---

## ✨ Features

* 🦸 Hero section with ghost **#7** watermark
* 👤 Biography and personal information card
* 📅 Animated career timeline
* 🏆 Trophy cabinet showcasing major honours
* 💬 Inspirational quote section
* 🌍 World records and achievements
* 📱 Fully responsive design
* 🎞️ Scroll-triggered animations
* 🖱️ Smooth scrolling navigation
* 🎨 Dark CR7-inspired theme

---

## 🛠️ Technologies Used

* **HTML5** — Semantic structure
* **CSS3** — Styling, Grid, Flexbox, animations
* **JavaScript** — IntersectionObserver API
* **Google Fonts**

  * Bebas Neue
  * Playfair Display
  * Inter
