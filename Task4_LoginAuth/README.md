\# AUTH\_FLOW 🔐 — Login Authentication System



\### Secure Register, Login \& Dashboard Application



A complete authentication system built using \*\*HTML, CSS, and Vanilla JavaScript\*\* featuring user registration, login validation, password strength checking, and a secured dashboard.



\---



\## 🌐 Live Demo



🔗 https://sebastinajoe.github.io/OIBSIP/Task4\_LoginAuth/



\---



\## 📌 Project Overview



\*\*AUTH\_FLOW\*\* is a modern login authentication system developed as part of the \*\*Oasis Infobyte Internship Program (OIBSIP)\*\*.



The application includes registration, login, and dashboard pages within a single HTML file. It provides real-time validation, password strength indicators, toast notifications, and secure access control using JavaScript.
