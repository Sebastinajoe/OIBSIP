\# TASK\_FLOW ✅ — To-Do Web App



\### Modern Task Management Application



A feature-rich To-Do web application built using \*\*HTML, CSS, and Vanilla JavaScript\*\* with task tracking, progress monitoring, editing, timestamps, and responsive design.



\---



\## 🌐 Live Demo



🔗 https://sebastinajoe.github.io/OIBSIP/Task3\_TodoApp/



\---



\## 📌 Project Overview



\*\*TASK\_FLOW\*\* is a modern task management application developed as part of the \*\*Oasis Infobyte Internship Program (OIBSIP)\*\*.



The application allows users to add, edit, complete, and delete tasks while tracking overall progress through live statistics and a dynamic progress bar.



\---



\## ✨ Features



\* ➕ Add new tasks instantly

\* ✅ Mark tasks as completed

\* ✏️ Edit existing tasks using a modal popup

\* 🗑️ Delete tasks individually

\* 🧹 Clear all pending or completed tasks

\* 📋 Separate Pending and Completed sections

\* 📊 Live statistics (Total, Pending, Completed)

\* 📈 Dynamic progress bar

\* 🕐 Added and completed timestamps

\* 🎞️ Smooth slide-in animations

\* 📅 Live date display

\* ⌨️ Keyboard shortcuts support

\* 🔒 XSS-safe task rendering

\* 📱 Fully responsive design
