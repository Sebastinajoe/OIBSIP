## ✅ Task Overview

### 🧮 Task 1 — Calculator

**Folder:** `Calculator/`
**Live Demo:** https://sebastinajoe.github.io/OIBSIP/Calculator/

A modern calculator application built with HTML, CSS, and JavaScript.

#### Features

* Basic arithmetic operations
* Percentage and sign toggle
* Calculation history panel
* Keyboard support
* Backspace support
* Division-by-zero error handling
* Dark neon UI theme
* Responsive design

**Technologies:** HTML5, CSS3, Vanilla JavaScript

---

### ⚽ Task 2 — Tribute Page

**Folder:** `TributPage/`
**Live Demo:** https://sebastinajoe.github.io/OIBSIP/TributPage/

A tribute page dedicated to **Cristiano Ronaldo**, celebrating his life, achievements, and football legacy.

#### Features

* Hero section with ghost #7 watermark
* Biography section
* Career timeline
* Trophy cabinet
* Quote section
* World records
* Responsive design
* Scroll-triggered animations

**Technologies:** HTML5, CSS3, JavaScript, IntersectionObserver API
